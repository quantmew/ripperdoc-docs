## 环境要求

- Python 3.10 或更高版本
- pip 或 uv 包管理器

## 安装方式

### 使用 pip（推荐）

```bash
pip install ripperdoc
```

### 使用 uv

```bash
uvx tool install ripperdoc
```

### 从源码安装

```bash
git clone https://github.com/anthropics/ripperdoc.git
cd ripperdoc
pip install -e .
```

## API Key 配置

Ripperdoc 需要你使用的 LLM 提供商 API key。请设置对应的环境变量：

### Anthropic Claude（默认）

```bash
export ANTHROPIC_API_KEY="your-api-key"
```

### OpenAI

```bash
export OPENAI_API_KEY="your-api-key"
```

### DeepSeek

```bash
export DEEPSEEK_API_KEY="your-api-key"
```

### Google Gemini

```bash
export GEMINI_API_KEY="your-api-key"
```

## 验证安装

```bash
ripperdoc --version
```

你应该能看到版本号。然后运行：

```bash
ripperdoc
```

这会启动一个交互式会话。输入消息即可开始与 AI 对话。

## Shell 自动补全（可选）

Ripperdoc 支持 bash 和 zsh 自动补全：

```bash
# For bash
eval "$(ripperdoc --completion bash)"

# For zsh
eval "$(ripperdoc --completion zsh)"
```

把对应的行加入你的 shell rc 文件（`.bashrc` 或 `.zshrc`）即可永久启用补全。
